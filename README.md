# 💳 Transaction Reconciliation System

A production-style **Payment Reconciliation Tool** built using **Python & Pandas**.

---

## 🚀 Features

* Generate synthetic transaction data
* Generate settlement data
* Detect mismatches
* Identify duplicates
* Generate summary report

---

## 🛠️ Technologies

* Python
* Pandas
* NumPy
